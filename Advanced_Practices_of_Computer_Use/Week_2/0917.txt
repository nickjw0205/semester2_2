숫자는 오른쪽 정렬, 텍스트는 왼쪽정렬

날짜는 "-"(2018-09-17)로도 쓸수잇지만 "/"(2018/09/17)으로도 쓸수있음.

오늘날짜 --> Ctrl + ;
지금시간 --> Ctrl + Shift + ;

사용자지정목록만들기.
파일 -> 옵션 -> 고급 -> 사용자지정목록
-> 달력모양누르기 -> 목록 부분 드래그.

오후 12시 -> 0.5로 저장됨. 시간은 이런식으로 비율로 저장이 됨.

서식만 복사.
복사할 서식 드래그 -> 홈탭에 서식복사 클릭 -> 복사할 부분선택!

근무시간 * 최저임금 할때 시간은 24로 나눈 값으로 저장되어 있기때문에 
셀*24*7530을 해야한다.

통화와 회계의 차이 -> \표시의 위치의 차이
회계 -> \은 왼쪽에 숫자는 오른쪽에, 0원을 입력하면 "-"으로 표시가 됨.
통화 -> 둘다 오른쪽에, 0원을 입력하면 0원으로 입력이 됨.

*텍스트 나누기(규칙성이 존재해야함.)
데이터 탭 -> 텍스트 나누기
"." 으로 숫자와 글자가 나뉘므로 구분기호로 나누기를 선택.
"."가 없으므로 기타에 "."을 입력한다.

테두리는 홈탭 글꼴에 밭전자 모양을 클릭하여 수정가능.

홈탭 -> 맞춤 -> 병합하고 가운데 맞춤 하면 이쁘게 병합가능.

특정셀만 선택할때 -> 찾기및선택 -> 이동옵션 -> 빈셀만 선택
-> 오른쪽마우스 -> 삭제 -> 행삭제 ===> 이렇게 옆에 셀을 같이 삭제할 수 잇음

Ctrl + Enter --> 값복사.

특정 문자를 바꿀때 -> 찾기및선택 -> 바꾸기 -> 바꿀내용 입력.

서식복사를 더블클릭하면 여러번 복사할 수 있음. 
--> 여러개의 서식을 복사해 한번에 많이 붙여넣을 수도 있음.
