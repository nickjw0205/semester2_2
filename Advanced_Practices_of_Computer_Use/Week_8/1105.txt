순번을 매길때 행을 삭제하거나 삭제를 하면 번호를 업데이트를 
해줘야한다는 불편함이 존재한ㄴ다.
==> 함수를 이용하자!

순번은 counta를 통해 옆에 셀의 내용을 선택 해서 첫행을 고정시키고 
드래그를 하면 셀이 업데이트 되면서 개수가 점점 늘어나 순번이 나타난다.
이러면 행을 삭제해도 순번이 어긋나지 않게 된다.

row(): 특정 셀의 행번호를 알고싶을 때
column(): 특정 셀의 열번호를 알고싶을 때
이걸 사용해서 순번을 구할수도 잇음요
괄호안에 셀의 위치를 입력하면 그 위치의 번호를 주고 안입력하면
현재 수식이 입력되는 셀의 위치를 알려준다.

홈텝에 편집에 시그마모양을 누르면 평균이있다.

순위를 구하는거는 rank (그러나 느낌표가 붙어있다! 되도록 사용하지 말라!)

rank.avg => 같은 점수는 평균 (2,3등은 2.5등으로)
rank.eq => 같은 점수는 같은 등수

=if(셀 >= ~~,TRUE일때,FALSE일때) 이런식으로 가능함
함수 인수 창에서 다른 함수 인수 창으로 가고싶을때는 수식입력줄에 이동하고싶은 함수를 누르면 된다

논리함수 => and(), or() : 2개이상의 조건을 쓸때는 꼭필요해유

빈칸은 눈에 보이지 않는 문자가 있는것
빈셀은 아무것도 없는것.

문자열 조작 함수
left(): 왼쪽을 기준으로 주어진 개수만큼 부분 추출을 함
mid(): 문자열의 중간부터 특정 문자열을 부분 추출
right(): 오른쪽을 기준으로 주어진 개수만큼
